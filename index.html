Disclaimer

⚠️ This service is built on limited free licenses for demo purposes only. It leverages the **Financial Modeling Prep (FMP) API**, which has limited securities coverage. Performance and data availability may be restricted.

This project is implemented using **FastAPI (Python) for backend processing**, **JavaScript for frontend logic**, and **FMP API for financial data retrieval**. The risk calculations, including **Value at Risk (VaR)**, are performed using **NumPy and SciPy**. The interface is built with **HTML, CSS, and JavaScript** for real-time interactivity.

Why This Matters

This simple project highlights the **importance of smart investment portfolio composition** and how dramatically it can impact risk management. By diversifying assets strategically, an investor can **significantly lower the overall portfolio VaR**, reducing potential losses under normal market conditions.
