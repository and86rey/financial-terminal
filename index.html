Disclaimer

This service is built on limited free licenses for demo purposes only. It uses the Financial Modeling Prep (FMP) API, which has restricted securities coverage. Performance and data availability may be limited.

The project is developed using FastAPI (Python) for backend processing, JavaScript for frontend logic, and the FMP API for financial data retrieval. Risk calculations, including Value at Risk (VaR), are performed with NumPy and SciPy. The interface is built with HTML, CSS, and JavaScript to ensure real-time interactivity.

Why This Matters

This project demonstrates the impact of smart investment portfolio composition on financial risk management. A well-diversified portfolio can significantly reduce overall Value at Risk (VaR), limiting potential losses under normal market conditions. Understanding and optimizing risk allocation is a key factor in making informed investment decisions.
