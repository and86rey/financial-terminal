Portfolio Risk Analyzer

About This Project

This web application allows users to compose a simple investment portfolio with up to five securities and analyze its risk using Value at Risk (VaR) metrics. Users can search for securities by ISIN, company name, or stock ticker, assign weight percentages, and calculate the total portfolio VaR.

This project highlights the importance of smart portfolio composition in mitigating financial risk. By comparing single-security VaR values against a diversified portfolio VaR, users can better understand how asset selection impacts risk exposure.

Disclaimer

This tool is provided for educational and demonstration purposes only. The VaR calculations are based on historical price data and do not constitute financial advice. Data is retrieved using the FMP API, which has limited securities coverage.

This project is built using JavaScript for the frontend, Python with FastAPI for the backend, and hosted on GitHub Pages with the backend running on Render.

Note: The free version of this tool limits the portfolio to five securities.
